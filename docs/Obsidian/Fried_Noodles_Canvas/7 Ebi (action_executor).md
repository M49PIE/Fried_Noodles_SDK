1. Совершаем действия в окружении через локомоции [[7.1 Locomotion]]. Локомоции имеют свой вектор [[7.2 Vector]]. 
2. Действия могут поменять фрустрацию и окружение. 
3. Если несколько раз не получилось, то используем речь. 
4. Спустя определенное число циклов запускаем консолидацию [[7.3 Consolidation]] чтобы сохранить накопленную информацию. 
5. Если нет знакомых стимулов то происходит полевое поведение [[7.4 Field Action]].
