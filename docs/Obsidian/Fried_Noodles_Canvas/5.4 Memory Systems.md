Уровни памяти (Алексей Сорока) - модель, разделяющая память на несколько уровней

Уровни памяти: 
1. актуальная - используется для удовлетворения актуальной потребности. 
2. основная [[5.11 Persistent Memory Layer]] - сохраненная на случай если потребуется снова использовать использованное ранее поле. 
3. резервная - если поле в течении долгого времени не актуализируется. используется если ничего больше не помогает.
