На основе восприятия достраивается и сохраняется поле [[5.1 Field]] через атрибуцию [[5.2 Attribution]]. Это поле представляет собой схематическую копию реальности в сознании агента. 
В поле будет строиться ходологическое пространство [[5.3 Hodological Space]] на основе актуальных потребностей.
Храниться поле в уровнях памяти [[5.4 Memory Systems]] и разделяется между собой границами [[5.5 Boundary]] для оптимизации.
Объекты пространства хранятся в семантических примитивах [[5.6 Semantic Primitives]]. Семантические примитивы объекты имеют свое название (как правило), атрибуты, координаты и валентность [[5.7 Valence]]. Она может быть положительная [[5.8 Positive Valence]], негативная [[5.9 Negative Valence]], нейтральная [[5.10 Zero Valence]]. Семантические примитивы могут быть изучены через  оперантное обусловливание, викарное научение или посредством остенсивного обучения [[5.12 Ostensible Learning]] через иннативный словарь [[5.13 Innative Dictionary]]. Если объект не имеет название, то он его придумает через генерацию названий [[5.14 Generated Naming]].
Взаимодействие субъектов с объектами хранится как семантическая тройка [[5.15 Semantic Triple]]. Семантическая тройка может быть изучена через оперантное обусловливание, викарное научение или посредством остенсивного обучения через нарративное обучение [[5.16 Narrative Dictionary]].
